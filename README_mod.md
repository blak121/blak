
# شروحات السوري

## معلومات حول الأداة
هذه الأداة تهدف إلى تحسين الهجمات على حسابات فيسبوك باستخدام واجهة مستخدم محسنة من خلال مكتبة `rich`.

## طريقة الاستخدام:
1. استنساخ المشروع:
   ```bash
   git clone https://github.com/YOUR_USERNAME/aFriDiy
   ```

2. الانتقال إلى مجلد الأداة:
   ```bash
   cd aFriDiy
   ```

3. تشغيل الأداة:
   ```bash
   python afD.py
   ```

## المتطلبات:
- Python 3.6 أو أعلى.
- مكتبات `requests` و `rich`.
